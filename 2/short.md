
# Краткий конспект пройденного
### Вывод информации
```
fmt.Print() —  выведет на консоль одной строкой без пробелов.
    
fmt.Println() — выведет на разных строках.
    
fmt.Println(число + число) — сложили два числа и вывели сумму (тут + — это математическое сложение).
    
fmt.Println("строка" + "пробел" + "строка") —  склеили строки через пробел (тут + — это конкатенация/склеивание).
```
### Переменные
```
var a int = 7 — объявили и инициализировали переменную а (обозначили тип данных и присвоили значение).
    
var a = 7 — объявили и инициализировали переменную а (без обозначения типа данных, с присвоением значения).
    
a := 7 — объявили и инициализировали переменную а (без обозначения типа данных, с присвоением значения).
    
a = a + 8 — увеличение переменной а на 8.
    
i++ — инкрементация увеличивает i на единицу.
    
i-- — декрементация уменьшает i на единицу.
    
123456 % 10 = 6 — деление с остатком позволяет получить последнюю цифру числа.
```
### Основные типы данных
```
int – целое число от -2147483648 до 2147483647;
string – строка;
float32 (float64)– число с дробной частью (вещественные числа);
rune – символ;
bool – специальный тип, принимает только два значения: true или false.
```
##### ​​​​​Конвертация
```
var s float64
fmt.Scan(&s) // считать число

var a  = int(s) // перевести (конвертировать) s в целое число

var d float64 = float64(a) // в вещественное число

var d float64 = 2.9 // записывается через точку

var n int = int(d) // если надо взять целую часть числа
```
### Пакет math
```
var b float64 = math.Abs(b) // модуль числа

var a float64 = math.Pow(b, 2) // возводим переменную b в степень 2

var a float64 = math.Sqrt(b) // корень квадратный

var b float64 = math.Min(a, b) // минимум из чисел а и b (для максимума math.Max(a, b))
```
